# JavaScript Registration Validation (Window API)

This project demonstrates input validation using JavaScript's Window API
(`prompt` and `alert`).

## Features
- Full Name validation (minimum of two words)
- Email format validation
- Strong password validation
- Password confirmation check
- Age restriction (18+)

## Technologies Used
- JavaScript (Window API)

## How to Run
1. Clone the repository
2. Open `index.html` in a browser
3. Follow the prompt instructions
